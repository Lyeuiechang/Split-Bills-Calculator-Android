# Split-Bills-Calculator-Android

## 拆帳計算機 (原 AA 制計算機)

讓聚會拆帳變超容易

特色：
- 支援從 1 至 6 人拆帳計算
- 可套入服務費率
- 詳細的計算公式過程
- 可幫每個人取綽號方便辨識

最大特色：
- 基於 Android Studio + Jetpack Compose 打造，支援原生 Android 多項特色：Material Design 3、暗色模式

相較於 AA 制計算機的更動：
- 每個人輸入區有累加按鈕
